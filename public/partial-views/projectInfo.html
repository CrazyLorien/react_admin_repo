App consists from two parts - client and admin.

Client area

Client can create order for barber mall. There are for stages when youser should give their desigion: choose barber, choose day and time, enter their own name and finally accept order. When user accept order it will email notification with confirmaion and order info. Also user can login via social network. For now we choose 2 systems (vk - for russian area, and face book for the other world). If user log in with social media he can add reviews(text and photo).If user upload photo - he can see it in site gallery. Non auth users can only see it.

Admin area

Admin can add new barber to shop. Administrate order requests (edit, update, delete) Also we add feature of adding reviews to admin area because we don't want create many roles for presentation site(but if somebody buy it we can add roles for admin, and users login via mail.
